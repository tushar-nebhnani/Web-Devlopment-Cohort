# Tushar Nebhnani - Resume

A professional HTML resume showcasing expertise in full-stack development, data science, and AI/ML.

## Overview

This project is a responsive HTML resume that highlights professional experience, skills, projects, education, and contact information. The resume is built with semantic HTML5 and styled CSS for a clean, professional presentation.

## Features

- **Responsive Design**: Mobile-friendly layout that works on all devices
- **Professional Styling**: Clean, minimalist design with custom CSS
- **Interactive Elements**: Hover effects on links and navigation
- **Structured Sections**:
  - Personal Information & Professional Summary
  - Skills (Programming Languages, Web Development, Databases, Data Science, Version Control)
  - Work Experience
  - Projects (E-commerce Predictive Dashboard, Retail Analysis Dashboard)
  - Education
  - Contact Information

## Skills Highlighted

- **Programming Languages**: Python, JavaScript, SQL
- **Web Development**: React, HTML5, CSS3, Tailwind CSS, FastAPI, Node.js
- **Databases**: MongoDB, MySQL
- **Data Science**: Machine Learning, Data Cleaning, Statistical Analysis, EDA
- **Version Control**: Git & GitHub

## Projects Included

1. **Olist ECommerce Predictive Dashboard** - ML + Web project with FastAPI backend and React frontend
2. **Retail Analysis Dashboard** - Interactive Streamlit dashboard for retail analytics

## How to Use

1. Open `index.html` in your web browser
2. View your professional resume with all sections and projects
3. Click on links in the contact section to reach out
